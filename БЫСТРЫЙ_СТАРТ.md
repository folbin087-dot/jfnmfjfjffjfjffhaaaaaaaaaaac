## 🚀 КАК ЗАПУСТИТЬ

### 1. Скомпилировать

```bash
cd "External 2/bycmd"
./build.bat
```

### 2. Запустить игру

1. Откройте Standoff 2
2. Зайдите в любой матч
3. Откройте меню чита
4. Включите ESP (Box, Name, Health, Skeleton)

### 3. Проверить

Должны появиться:

- ✅ Рамки вокруг врагов
- ✅ Имена над головами
- ✅ Зелёная полоска здоровья
- ✅ Скелет игрока
- ✅ Расстояние в метрах

## 🐛 ЕСЛИ НЕ РАБОТАЕТ

### Быстрая проверка:

1. Подождите 5-10 секунд после входа в матч
2. Убедитесь, что ESP включён в меню
3. Проверьте версию игры (должна быть 0.38.0)

### Отладка:

Добавьте в `visuals.cpp` после строки 620:

```cpp
static int debug_frame = 0;
if (debug_frame++ % 60 == 0) {
    FILE* f = fopen("/sdcard/esp_debug.txt", "a");
    if (f) {
        fprintf(f, "PlayerManager: 0x%llx, playersList: 0x%llx\n",
                PlayerManager, playersList);
        fclose(f);
    }
}
```

Перекомпилируйте и проверьте файл:

```bash
adb pull /sdcard/esp_debug.txt
cat esp_debug.txt
```

Если видите нули — проблема с оффсетами или версией игры.
