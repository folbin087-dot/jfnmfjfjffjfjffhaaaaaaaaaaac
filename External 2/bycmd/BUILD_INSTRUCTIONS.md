# Инструкция по сборке bycmd v0.37.1

## Требования

### Системные требования

- **Windows 10/11** (для сборки)
- **Android NDK r29**
- **CMake 3.18+** (включен в проект)
- **Ninja** (включен в проект)

### Настройка окружения

1. Скачайте и установите **Android NDK r29**
2. Отредактируйте `build.bat`, указав путь к NDK:
   ```batch
   set "NDK=D:\android-ndk-r29"
   ```

## Сборка проекта

### Автоматическая сборка

```batch
cd 0.37.0\External\bycmd
build.bat
```

### Результат сборки

После успешной сборки в папке `libs` появятся:

- `libs/arm64/bycmd_arm64` - для ARM64 устройств
- `libs/x86-64/bycmd_x86-64` - для x86-64 устройств/эмуляторов

## Использование

### Запуск

1. Скопируйте соответствующий бинарник на Android устройство
2. Дайте права на выполнение: `chmod +x bycmd_arm64`
3. Запустите: `./bycmd_arm64`

### Управление

1. **Активация меню**: Тап по полупрозрачной панели внизу экрана
2. **Навигация**:
   - **Visuals** - настройки ESP
   - **Aimbot** - настройки автоприцеливания
   - **Settings** - системная информация
3. **Настройка Aimbot**:
   - Включите "Enable Aimbot"
   - Настройте FOV (рекомендуется 60°-90°)
   - Установите плавность (рекомендуется 3.0-7.0)
   - Выберите приоритет и кость для прицеливания

## Совместимость

### ⚠️ Важные замечания

- **Только для Standoff 2 v0.37.1**
- **Несовместимо с предыдущими версиями игры**
- **Требует root доступа** для работы с памятью процесса

### Поддерживаемые архитектуры

- **ARM64-v8a** (основная, 99% устройств)
- **x86-64** (эмуляторы, Intel устройства)

## Устранение неполадок

### Проблемы сборки

- **"NDK not found"**: Проверьте путь к NDK в `build.bat`
- **"CMake not found"**: Используйте встроенный CMake из папки `tools`
- **Ошибки компиляции**: Убедитесь, что используете NDK r29

### Проблемы запуска

- **"Game not detected"**: Убедитесь, что Standoff 2 v0.37.1 запущен
- **Краши**: Проверьте совместимость версии игры
- **Нет доступа к памяти**: Требуются root права
